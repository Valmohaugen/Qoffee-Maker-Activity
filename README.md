## How to Use This Repository

### Clone the repository
- Download or clone this repository to your local machine.
- Or, just download the `Qoffee Maker Walkthrough.ipynb` notebook if you want to use it directly.

### Set up your environment
- Go to https://account.qbraid.com/
- Create a qBraid account or sign in with Google
- Press "Launch Lab" at the top
- Select the free version to use
- Press "ENVS" on the top right and then press "+ ADD"
- Search "Qiskit" and select "Qiskit v1.2.0"
- Install the environment
- Select "Python 3 [Qiskit v1.2.0]" under the "Notebook" section in the "Environment" tab

### Run the notebook
- Upload `Qoffee Maker Walkthrough.ipynb` using the up arrow in the top left below "View"
- Make sure you are in the "Python 3 [Qiskit v1.2.0]" kernel
- Open the notebook and follow the walkthrough
